# Theresa3rd-Bot-Web

## 简介
- 一个基于字节跳动前端框架 [Arco.Design](https://arco.design) 编写的适用于 [Theresa3rd-Bot](https://github.com/GardenHamster/Theresa3rd-Bot) 机器人后台管理页面

- 技术栈有 `vue3` + `Vite` + `TypeScript` + `Arco Design` + `Pinia` 等
